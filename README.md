# Json
Sk1er の JsonHolder の読み込み、保存

# Maven
Sk1er 様に JsonHolder.java を Maven 上に置いて良いと許可をいただきましたので
- Repository
```XML
  <repositories>
    <repository>
      <id>net.simplyrin</id>
      <name>api</name>
      <url>https://api.simplyrin.net/maven/</url>
    </repository>
  </repositories>
```

- Dependency
```XML
  <dependencies>
    <dependency>
      <groupId>net.simplyrin.json</groupId>
      <artifactId>Json</artifactId>
      <version>1.0</version>
    </dependency>
  </dependencies>
```
